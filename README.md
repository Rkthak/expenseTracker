## CashLens - Expense Tracker

CashLens is a smart expense tracking application built with **React JS**, **Tailwind CSS**, and **Chart.js**. It helps users track daily expenses, analyze spending habits, and visualize expense distribution through interactive charts. All data is stored in **localStorage**, ensuring persistence between sessions.

## 🚀 Features

### Expense Management

- Add expenses with:
  - Amount
  - Category
  - Date
  - Note

- View all expense entries
- Delete expenses instantly

### Category-wise Tracking

- Predefined categories:
  - 🍔 Food
  - ✈️ Travel
  - 🧾 Bills
  - 📦 shopping

- Track total spending per category
- Dynamic category-wise expense summary

### Filters & Sorting

- Filter expenses by category
- Filter by date
- Filter by amount range
- Search expenses easily
- Sort by:
  - Most Recent
  - Highest Amount

### Data Visualization

- Interactive Pie Chart using Chart.js
- Real-time category-wise expense distribution
- Automatic chart updates when expenses are added or deleted

### Data Persistence

- Uses browser localStorage
- Data remains available after page refresh or browser restart

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/Rkthak/expenseTracker.git
cd cashlens-expense-tracker
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

---

## 🎯 Usage

1. Add an expense with amount, category, date, and note.
2. View expenses in the expense list.
3. Apply filters and sorting options.
4. Monitor category-wise spending.
5. Analyze expenses through charts.
6. Data is automatically saved to localStorage.
